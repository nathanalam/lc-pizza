# Alam Pizza Group | Little Caesars

Welcome to the Alam Pizza Group | Little Caesars website project. This repository contains the source code for the website, which provides information about Family-owned Little Caesars restaurants in the Washington D.C. metro area.

## Project info

**URL**: https://littlecaesars.com (or your respective domain)

## How can I edit this code?

You can clone this repo and push changes.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I build and deploy this project?

To build the project for production, run:

```sh
npm run build
```

This will create a `dist` directory with the compiled assets, which can be deployed to any static site hosting provider (e.g., Vercel, Netlify, GitHub Pages, etc.).
